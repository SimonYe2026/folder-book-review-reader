# 转换后的夹具输出

这个文件夹保存已提交的转换器输出，用作测试夹具。

它不是普通用户的日常输出目录。正常的本地转换结果不应提交进源码仓库，除非它们足够小、内容公开，并且确实用于测试或 demo。

## English

## Converted Fixture Output

This folder contains checked-in converter output used as a fixture.

It is not routine user output. Normal local conversion outputs should stay outside committed source control unless they are intentionally small, public, and needed for tests or demos.
