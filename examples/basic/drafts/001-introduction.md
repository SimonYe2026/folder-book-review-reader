# Introduction

This is a small local reader demo.

It treats a folder of Markdown and TXT files like a book. You can search, move between chapters, adjust the layout, and collect review notes without changing the source files.

This sample includes a normal Markdown link to [project notes](https://example.com/review-reader) so the reader can verify that links are disabled by default.

## What to Try

- Search for `review`.
- Select this sentence and press `R`.
- Drag the left and right panel borders on a wide screen.

> The reader is intentionally local-first: it does not upload files and does not call an AI API.
