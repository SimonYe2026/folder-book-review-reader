# Review Workflow

The review board is the main feature.

When a passage needs attention, select text or use the paragraph review button. The reader creates a review item with the file path, title, block number, action, target, quote, and your instruction.

The exported `review.md` can be given to an AI assistant or used as a manual editing checklist.

Relative links such as [plain text sample](003-plain-text.txt) are also shown as disabled link placeholders in the generated reader.
